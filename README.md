# The Global Times

**Facts Before Narratives**

An autonomous, AI-powered digital newspaper delivering verified, multi-perspective journalism.

## Features

- **Daily Editions**: Fresh news published every day
- **Full Newspaper Sections**: World, Politics, Business, Technology, Science, Health, Culture, Opinion, Sports
- **Verification Transparency**: Every story includes source citations and confidence levels
- **Bias Analysis**: Framing differences are disclosed
- **Podcasts**: Daily audio briefings (coming soon)
- **Dark Mode**: Automatic system preference detection
- **Mobile Responsive**: Optimized for all devices

## Sections

| Section | Description |
|---------|-------------|
| World | International news and global affairs |
| Politics | Government, elections, and policy |
| Business | Markets, economy, and companies |
| Technology | Tech, AI, and innovation |
| Science | Research and discoveries |
| Health | Medicine and public health |
| Culture | Arts, entertainment, and trends |
| Opinion | Editorials and analysis |
| Sports | Scores and features |

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# View at http://localhost:4000
```

## Publishing

The newspaper is published via GitHub Pages. New articles are automatically deployed when pushed to the `main` branch.

## Technology

- **Static Site Generator**: Jekyll
- **Hosting**: GitHub Pages
- **Styling**: Custom CSS with classic newspaper aesthetics
- **JavaScript**: Vanilla JS for interactivity
